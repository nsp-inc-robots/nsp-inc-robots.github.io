01 Features 機能
  • 01

    Instant Link Preview

    インスタント・プレビュー

    Instead of opening a link in a new tab, display a preview window directly on the current page. Efficiency boosted: Check content without interrupting your flow.

    リンクを新しいタブで開く代わりに、現在のページ上に直接プレビューウィンドウを表示します。 今読んでいる記事を中断することなく、中身をサクッと確認できます。

  • 02

    Intuitive Activation (Customizable)

    直感的な起動方法（カスタマイズ可能）

    Supports activation by Alt / Ctrl / Shift + Click, or simply hovering over the link. Choose the style that fits your workflow for instant confirmation.

    Alt / Ctrl / Shift キーを押しながらのクリックや、リンクにマウスを乗せるだけ（ホバー）の起動に対応。 自分のスタイルに合わせた「一瞬の確認」を実現します。

  • 03

    Premium Multi-Window

    プレミアム・マルチウィンドウ

    Pin preview windows to keep them fixed, or adjust sizes (Small, Medium, Large) freely. Open multiple previews for effortless side-by-side comparison.

    プレビューウィンドウは、好きな場所にピン留めして固定したり、サイズ（小・中・大）を自由に調整したりできます。 複数のプレビューを並べて比較することも可能です。

  • 04

    Immersive Design

    没入感を高めるデザイン

    Featuring the signature NSP series dark theme and glassmorphism UI. Adjustable opacity (0.8–1.0) ensures a perfect balance between the preview and the main page.

    NSPシリーズ特有の、高級感あふれるダークテーマとグラスモルフィズムUIを採用。 不透明度の調整（0.8〜1.0）により、元のページとの視認性を両立させます。
