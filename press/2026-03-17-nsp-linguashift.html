NSP inc.、Chrome拡張機能「NSP LinguaShift」を公開
——普段のウェブ閲覧を、そのまま語学学習に変える新感覚アシスタント

NSP inc. Launches "NSP LinguaShift" Chrome Extension
— Turn Your Daily Browsing into Language Learning

NSP inc. は本日2026年3月17日、Google Chrome向け言語学習拡張機能「NSP LinguaShift（エヌエスピー・リングアシフト）」の公開を発表しました。本製品は、ユーザーが普段閲覧しているウェブページの日本語テキストを自動的に英語または韓国語へ置換することで、日常のブラウジングをそのまま語学学習の場へと転換します。全機能を制限なく、完全無料でご利用いただけます。

■ 開発の背景

語学学習における最大の障壁は「継続性」です。専用のアプリを開いたり、参考書を手に取ったりするには、毎回意志力と時間を必要とします。多くの学習者がこの「始めるハードル」を越えられずに挫折してきました。

NSP LinguaShiftは、すでに日常習慣として確立しているウェブ閲覧の中に、自然に語学学習を組み込みます。意識的に「勉強モード」に入ることなく、いつものニュースを読む時間が、そのまま英語・韓国語のインプット時間へと変わります。

製品のコンセプトは「『勉強する時間』を、『ウェブを見る時間』に変える。」です。

■ 主な機能

1. 高度な日本語解析エンジン
形態素解析ライブラリ「Kuromoji」を搭載し、日本語文の構造を解析した上で、学習効果の高い単語を選定して置換します。単純なランダム置換とは異なり、文脈を壊さない自然な変換を実現しています。

2. 英語・韓国語のダブル対応
ビジネスや資格試験に不可欠な「英語」と、K-POPやドラマ・旅行など文化的背景から需要が高まっている「韓国語（ハングル）」の両言語に対応しています。拡張機能のポップアップ上でワンタップ切り替えが可能です。

3. 調整可能な置換レベル（10〜80%）
ページ内の頻出単語ランキング上位から「何種類の単語を置換するか」をスライダーで自由に設定できます。語学学習の初期段階では10〜20%程度から始め、習熟度が上がるにつれて徐々に引き上げていくグラデーション学習が可能です。

4. 高品質TTS（音声合成）による発音確認
置換された単語をクリックすると、高品質な音声合成（Text-to-Speech）でネイティブに近い発音を即座に再生します。目で読むだけでなく、耳からも同時に学べるマルチモーダルな学習体験を提供します。

5. ホバーで確認できる文脈情報ツールチップ
置換された単語にマウスカーソルを合わせると、元の日本語・読み方・習得状況がポップアップ表示されます。ページを離れることなく意味確認ができるため、閲覧の流れを途切らせません。

■ 学習進捗の可視化

拡張機能のポップアップでは、「習得済み単語数」「連続学習日数（ストリーク）」「今日習得した単語数」の3つの指標をリアルタイムで確認できます。学習の継続を促すゲーミフィケーション要素として機能します。

■ その他の特徴

  • サイトごとのオン/オフ切り替え機能（無効にしたいサイトを個別指定可能）
  • 翻訳にMyMemory翻訳APIを使用（追加設定不要）
  • 学習データのリセット機能
  • 翻訳キャッシュクリア機能

NSP inc. today announced the release of "NSP LinguaShift," a language learning extension for Google Chrome. The tool automatically replaces portions of Japanese text on any webpage with English or Korean, transforming everyday web browsing into a language learning experience. All features are available completely free of charge.

■ Background

The biggest obstacle to language learning is consistency. Opening a dedicated app or picking up a textbook requires willpower and time — hurdles that cause many learners to give up. NSP LinguaShift solves this by embedding vocabulary acquisition directly into a habit users already have: reading websites.

The product concept: "Don't Study. Just Browse."

■ Key Features

1. Intelligent Morphological Analysis
Powered by the Kuromoji library, LinguaShift analyzes the structure of Japanese sentences to select and replace words where learning impact is highest — a meaningfully different experience from random substitution.

2. English & Korean Dual Support
Supports both English (essential for business and exams) and Korean (driven by K-pop and drama culture). Switch between languages with a single tap in the popup.

3. Adjustable Learning Intensity (10–80%)
A slider controls what percentage of high-frequency words on the page get replaced. Beginners can start at 10–20% and increase intensity as their vocabulary grows.

4. Crystal Clear Audio Playback (TTS)
Click any replaced word to hear it spoken aloud in high-quality text-to-speech. Learn with both eyes and ears simultaneously.

5. Instant Contextual Tooltip
Hovering over any replaced word surfaces the original Japanese, phonetic reading, and mastery status in a popup — no need to leave the page.

■ Learning Stats

The extension popup displays three real-time metrics: total words learned, current learning streak (days), and words learned today — adding a gamification layer that encourages consistent use.

■ Additional Features

  • Per-site enable/disable toggle
  • Uses MyMemory Translation API (no setup required)
  • Learning data reset
  • Translation cache clear
